# 📌 Portfolio 2024 - 前端視覺與互動探索實驗室

這個專案是我在 2024 年針對「前端動態特效」與「流暢視覺體驗」進行深度探索的實戰作品集。在這個階段，我專注於突破傳統網頁平版、靜態的佈局，大量實驗了 SVG 向量控制、3D 渲染與動態時間軸套件，奠定了後續開發高互動性應用的核心基石。

---

## 🛠️ 核心技術棧 (Tech Stack)

* **核心框架**：Vue 3 (Composition API / Options API 混用探索)
* **狀態管理**：Vuex 4 (負責跨組件非同步事件同步)
* **動態特效**：GSAP 3 (GreenSock Animation Platform)
* **3D 互動**：Three.js (基礎場景與滑鼠 Raycaster 互動探索)
* **UI 佈局**：Bootstrap 5 & Sass (RWD 響應式排版)
* **專案建構**：Vue CLI (Webpack 5)

---

## 🌟 核心互動亮點與 UX 優化思維

### 🎡 跨元件雙向滾輪控制系統 (3D Carousel + Navigation Sync)
* **優化動機**：在傳統的 3D 輪播元件中，使用者往往需要將滑鼠精準移至極小的「左/右箭頭」才能切換目標，操作體感不夠直覺。為了提升使用者體驗，我希望使用者能透過滑鼠滾輪直接在任意處無縫操作。
* **解決方案**：我重新設計了全頁面的事件攔截機制。當使用者在網頁任意處滾動滑鼠時，頁面會自動切換大分類；而當滑鼠移入 3D 輪播區域時，系統會自動切換監聽事件，改為旋轉 3D 卡片，提供流暢的無縫操作體驗。
* **技術突破 (Vuex 狀態管理)**：由於「滑鼠滾輪事件」、「全頁面大分類導覽列」與「Vue-Carousel-3D 元件」分屬不同的組件層級，為了解決複雜的跨組件狀態傳遞（Props / Emits 混亂問題），在此專案中我**首次導入 Vuex 進行全域狀態管理**。透過單一數據源（Single Source of Truth）來精準控制、同步各組件的動態切換狀態，成功克服了非同步事件流導致的畫面卡頓問題。

## 🧪 前端核心特徵與技術實驗室 (Lab Showcase)

在專案的研發過程中，我建立了多個獨立的技術實驗室（實驗頁面），用以驗證各種前端前端新技術的可行性：

| 實驗項目 | 技術核心 | 開發反思與優化（面試精華點） |
| :--- | :--- | :--- |
| **背景顏色變化效果** | ‧ Vue 3 雙向綁定 (`v-model`)<br>‧ 動態樣式綁定 (`v-bind:style`) | **實踐客戶端即時運算**：捨棄傳統將資料送回伺服器處理的模式，利用 Vue 響應式原理在瀏覽器端即時捕捉 `<input type="range">` 數值並動態生成 CSS 漸層字串，達成 0 延遲的流暢互動體驗，同時大幅降低網路頻寬與伺服器的運算耗損。 |
| **GSAP 形狀變換** | ‧ SVG Path 向量路徑控制<br>‧ Vue 響應式陣列切換 | **第三方套件授權評估與向量資產獨立提取**：<br>1. **技術探勘**：原計畫引入 GSAP MorphSVG 實現形狀補間動畫，評估後發現該功能屬付費授權，為兼顧開發成本，隨即將方案優化為「數據驅動路徑」機制。<br>2. **資產工程化流程**：自主透過 `SVGRepo` 檢索圖形，導入 Figma 進行路徑節點優化與畫布尺寸標準化（以確保切換時不跑位），最終以 SVG 格式導出並解析 XML 文本，手動提取精準的 `d` 屬性路徑字串綁定於 Vue `reactive` 狀態中，成功實現輕量化的幾何圖形即時切換系統。 |
| **GSAP 撲克隨機換牌** | ‧ SVG 坐標系轉換與幾何定位<br>‧ GSAP 3D 翻轉與 Timeline 洗牌 | **偽 3D 視覺欺騙演算法與資產原點校正**：<br>1. **資產重心校正**：手動重構 SVG 向量路徑，將花色 `d` 屬性出發點強制校正為幾何中心 (0,0)，確保在 SVG 畫布中利用 `transform` 縮放與旋轉時重心完全精準。<br>2. **偽 3D 翻轉動態系統**：利用 CSS `perspective` 與 GSAP 控制 `rotationY`。為了優化效能，設計了**視覺欺騙邏輯**：當卡片旋轉至 90 度（即垂直於視角、厚度趨近於零的瞬間），透過腳本即時更換數值並切換正反面元件的顯隱狀態，再繼續完成 90 到 180 度的翻轉，以輕量級的 2D 渲染完美模擬出極具立體感的 3D 翻牌特效。 |

## 📈 2024 專案的迭代與成長意義

回頭檢視 `portfolio-2024`，它完整記錄了我從傳統靜態網頁邁向「高互動性前端工程師」的轉折點。雖然當時環境仍建構在較重的 Webpack (Vue CLI) 之上，且有部分技術（如 GSAP 付費外掛）未能如願落地，但我在這個專案中培養的：
1. **客戶端資料即時運算的優化思維**。
2. **Figma 與 SVG 程式碼串接的資產加工能力**。
3. **跨組件複雜事件流（Vuex）的管理經驗**。

這些寶貴的自學與解題經驗，皆成為我日後在 `portfolio-2025` 乃至後續現代化前端開發（如 Vite, WebGL 等）中最扎實的養分。
